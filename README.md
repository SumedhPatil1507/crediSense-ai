# 💳 CrediSense AI

An AI-powered **Credit Risk Scoring System** that predicts loan default probability and provides explainable financial decisions.

🌐 **Live App:** [https://credisense-ai-uzzvdsxuuxdbocfwxhmqcc.streamlit.app/](https://credisense-ai-uzzvdsxuuxdbocfwxhmqcc.streamlit.app/)

---

## 🚀 Features

- 🔍 Loan Risk Prediction (ML Model - LightGBM)
- 📊 Interactive Data Analysis Dashboard
- 🧠 Explainable AI (SHAP)
- 💰 Business Profit Simulation
- 🤖 AI Chatbot Assistant

---

## 🛠️ Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn, LightGBM
- SHAP (Explainability)
- Streamlit (Frontend)

---

## 📊 Model Performance

- F1 Score: High performance on imbalanced data
- ROC-AUC: Strong separation between risky and safe applicants

---

## 💡 Business Value

- Automates loan approval decisions
- Reduces default risk
- Simulates profit/loss for financial institutions

---

## 🚀 Hosting on Streamlit Cloud

1. Push your code to GitHub
2. Go to [https://streamlit.io/cloud](https://streamlit.io/cloud) and sign in
3. Click **New app** → select your repo, branch, and set main file path to `app/app.py`
4. Click **Deploy**

---

## 📦 Run Locally

```bash
# Install dependencies
pip install -r requirements.txt

# Start the app
streamlit run app/app.py
```
